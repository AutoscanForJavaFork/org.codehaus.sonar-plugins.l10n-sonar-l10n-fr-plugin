L'utilisation de convention de nommage est l'un des points clé permettant à une équipe de collaborer efficacement. Cette règle permet de vérifier que les noms de type paramétré respectent l'expression régulière définie sur cette règle.

Le code suivant illustre cette règle lorsque l'expression régulière est "^[A-Z]$": 

class MyClass<TYPE> { // Non-Compliant
  <TYPE> void addAll(Collection<TYPE> c) { // Non-compliant
  }
}

public class MyClass<T> { // Compliant
  <T> void addAll(Collection<T> c) { // Compliant
  }
}
