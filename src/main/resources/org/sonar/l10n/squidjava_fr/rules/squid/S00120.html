L'utilisation de convention de nommage est l'un des points clé permettant à une équipe de collaborer efficacement. Cette règle permet de vérifier que les noms de package respectent l'expression régulière définie sur cette règle.

Le code suivant illustre cette règle lorsque l'expression régulière est "^[a-z]+(\\.[a-z][a-z0-9]*)*$": 

package org.Example; // Non-Compliant

package org.example; // Compliant
