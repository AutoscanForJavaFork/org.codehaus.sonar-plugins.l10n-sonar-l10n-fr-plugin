L'utilisation de convention de nommage est l'un des points clé permettant à une équipe de collaborer efficacement. Cette règle permet de vérifier que les noms de variable locale respectent l'expression régulière définie sur cette règle.

Le code suivant illustre cette règle lorsque l'expression régulière est "^[a-z][a-zA-Z0-9]*$": 

public class MyClass {
  public void method() {
    int FIRST; // Non-Compliant
    int second; // Compliant
  }
}
