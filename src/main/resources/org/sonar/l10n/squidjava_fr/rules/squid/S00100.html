L'utilisation de convention de nommage est l'un des points clé permettant à une équipe de collaborer efficacement. Cette règle permet de vérifier que les noms de méthode respectent l'expression régulière définie sur cette règle.

Le code suivant illustre cette règle lorsque l'expression régulière est "^[a-z][a-zA-Z0-9]*$": 

public class MyClass {
  private int myProperty;

  public int getMyProperty() { // Compliant
    return myProperty;
  }

  public void SetMyProperty(int value) { // Non-Compliant
    myProperty = value;
  }
}
