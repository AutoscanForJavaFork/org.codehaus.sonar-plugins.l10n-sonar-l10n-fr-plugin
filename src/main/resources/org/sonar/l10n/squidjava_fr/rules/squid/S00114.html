L'utilisation de convention de nommage est l'un des points clé permettant à une équipe de collaborer efficacement. Cette règle permet de vérifier que les noms d'interface respectent l'expression régulière définie sur cette règle.

Le code suivant illustre cette règle lorsque l'expression régulière est "^[A-Z][a-zA-Z0-9]*$": 

public interface MyFirstInterface {...} // Compliant
public interface mySecondInterface {...} // Non-Compliant
